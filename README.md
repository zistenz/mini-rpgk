# Mini-rpgk

### Saját készítésű és lefordított mini rpg-k

A mini-rpgk az utóbbi években, évtizedben kezdtek elterjedni igazán, népszerűségüket az OSR (*old school renaissance*) mozgalom felfutásának is köszönhetik. Ezek a játékok néhány oldalon kínálnak egy olyan egyszerű szabályrendszert, ami jellemzően egy célzott világ-, vagy helyzetleíráshoz tartoznak, gyorsan elsajátíthatók és kitűnően alkalmasak akkor, ha csak néhány óránk van játszani (akár pl. utazás közben is). Viszonylagos egyszerűségük ellenére azért használhatók hosszabb távon is.

Ezen az oldalon az általam készített szerepjáték fordításokat és kisebb játékokat találhatod.
